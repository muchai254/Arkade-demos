## Boarding Address Demo

Demonstrates:

1. converting mnenomic phrase to 64-byte seed
2. deriving bip32 node at path `m/86'/0'/0'/0/0`
3. extracting public key from bip32 node
4. fetching public key from Arkade operator
5. constructing Arkade boarding tapscript
6. encoding as bech32m taproot address
7. validating against the boarding address generated by the Arkade SDK

Related: [onchain-address](https://github.com/arkade-os/demos/tree/master/onchain-address)
