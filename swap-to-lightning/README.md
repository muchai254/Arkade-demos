## Swap to Lightning Demo

Demonstrates:

1. setting up a mnemonic identity
2. connecting to operator + delegate and extracting necessary parameters
3. generating a delegated user address
4. fetching balance
5. validating and fetching invoice from LUD-16 address
6. verifying invoice amount against Boltz limits
7. creating a Boltz Arkade > Lightning swap
8. constructing and sending a transaction to the swap address, sweeping remaining funds to self

### TypeScript

`pnpm install && pnpm dev`
