## Arkade Address Demo

Demonstrates:

1. converting mnenomic phrase to 64-byte seed
2. deriving bip32 node at path `m/86'/0'/0'/0/0`
3. extracting public key from bip32 node
4. fetching public key from Arkade operator
5. generating default + delegate tapscripts
6. encoding both addresses using `ArkAddress`
7. validating against the addresses generated by the Arkade SDK

Related: [boarding-address](https://github.com/arkade-os/demos/tree/master/old/boarding-address)

### TypeScript
`pnpm install && pnpm dev`

### Rust
`cargo run`
