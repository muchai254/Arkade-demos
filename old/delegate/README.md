## Delegate Demo

Demonstrates:

1. setting up mnenomic identity
2. connecting to operator and extracting its pubkey, forfeit address outscript, and unilateral exit timelock
3. connecting to delegate and extracting its pubkey, fee collection address and fee
4. constructing a delegated user address
5. fetching inputs received by the delegated address
6. constructing a signed delegation intent
7. constructing signed forfeit transactions
8. submitting a delegation request to the delegate

Related: [arkade-address](https://github.com/arkade-os/demos/tree/master/old/arkade-address)

### TypeScript
