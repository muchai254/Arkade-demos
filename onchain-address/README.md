## Onchain Address Demo

Demonstrates:

1. converting mnenomic phrase to 64-byte seed
2. deriving bip32 node at path `m/86'/0'/0'/0/0`
3. extracting public key from bip32 node
4. generating standard tapscript (keypath spend)
5. encoding as bech32m taproot address
7. validating against the onchain address generated by the Arkade SDK

Related: [boarding-address](https://github.com/arkade-os/demos/tree/master/boarding-address)

### TypeScript
`pnpm install && pnpm dev`

### Rust
`cargo run`
